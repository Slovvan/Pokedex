# Pokedex

The Pokemon Mobile App allows users to browse through a list of all Pokemon and mark their favorites. It also includes basic authentication for users to create an account and log in.

## Usage

1. Sign up or log in to access the app.
2. Browse through the list of all Pokemon.
3. Tap on a Pokemon to view its details.
4. Tap the "Favorite" button to mark a Pokemon as your favorite.
5. View your favorite Pokemon by navigating to the "Favorites" tab.

## Features

- Browse through a list of all Pokemon.
- View detailed information about each Pokemon.
- Mark Pokemon as your favorites.
- User authentication and authorization.
